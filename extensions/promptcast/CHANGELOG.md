# PromptCast Changelog

## [Initial Version] - 2025-12-19

### Added

- My AI Prompts command: Browse, search, and launch saved prompts
- Create AI Prompt command: Save new prompts with tags and variables
- Variable support using `{variableName}` syntax for dynamic prompt customization
- Deep link integration with Claude (claude.ai) and ChatGPT (chatgpt.com)
- Tag-based organization for prompt categorization
- Keyboard shortcuts for quick launching to AI platforms
- Local storage using Raycast LocalStorage for data persistence
- Variable input form for filling prompt placeholders before launch
- Multi-app selection for launching prompts to multiple AI platforms simultaneously
